# Security Policy

## Supported Versions

Currently, the `latest` version of Tagix is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in Tagix, please do NOT open a public issue.

Instead, please report it via privacy email or GitHub Security Advisories if enabled.

1. Describe the vulnerability.
2. Provide steps to reproduce logic.
3. We will review and respond as quickly as possible.
